## 一、项目是什么

**格物 · Agent 卷**是一套面向 agent 系统的动手学习系列，目标是从底层理解：

- agent harness
- tool use
- memory / state
- ReAct / Reflexion / planning
- sub-agent
- agent team
- multi-agent collaboration
- evaluation / trace
- MCP / A2A 等协议层

它延续大模型卷的理念：**看懂原理 -> 亲手从零实现**。

## 三、核心设计

Agent 卷不先从框架入手，而是先从最小 harness 入手：

```text
User Goal
  -> LLM decides next action
  -> Harness executes tool
  -> Tool observation returns to LLM
  -> LLM decides continue or finish
```

核心学习原则：

- 先 single-agent，再 multi-agent
- 先 harness，再框架
- 先 trace，再评价
- 先预算对齐，再谈性能提升
- 每个抽象都配可运行代码

## 五、Web 关系

现有 `web/` 是大模型卷的 14 章可视化演武场，入口和卷目录目前写在 `web/src/App.jsx`。

Agent 卷暂时不直接改动 Web 主入口，原因：

- 目前 Agent 卷还没有成熟的可视化 demo
- 贸然加入导航会出现“有卷名、无演武场”的半成品
- Agent 卷的 Web 需要设计新的交互：trace、tool call、observation、并行 agent 时间线

Web 接入计划见：
